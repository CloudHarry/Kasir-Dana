# 📱 Kasir DANA — Panduan Instalasi Android

## Cara Install di Android (Seperti APK)

### Metode 1: Via File Manager (Lokal)
1. Salin folder `kasir-dana/` ke HP Android Anda
2. Buka **Google Chrome** di HP
3. Ketik di address bar: `file:///storage/emulated/0/Download/kasir-dana/index.html`
4. Tap ikon **⋮ (tiga titik)** di pojok kanan atas
5. Pilih **"Tambahkan ke layar utama"** / **"Add to Home screen"**
6. Tap **"Tambah"** → Ikon Kasir DANA muncul di Home Screen!

### Metode 2: Via Web Server Lokal (Disarankan)
1. Upload semua file ke hosting (Netlify, GitHub Pages, dll.)
2. Buka URL di Chrome Android
3. Tap **⋮ → "Tambahkan ke layar utama"**
4. App bisa dipakai **OFFLINE** setelah pertama kali dibuka!

### Metode 3: Via USB / ADB
1. Salin ke storage HP
2. Ikuti langkah Metode 1

## 📞 Cara Pakai

### Transaksi:
1. Buka tab **Kasir**
2. Cari / pilih produk → Tap untuk tambah ke keranjang
3. Tap ikon 🛒 di pojok kanan atas
4. Tap **Bayar Sekarang**
5. Masukkan uang diterima / tap nominal cepat
6. Tap **Proses Transaksi** → Struk muncul!

### Tambah Produk:
1. Buka tab **Produk**
2. Tap tombol **＋** (biru, pojok kanan bawah)
3. Isi nama, harga, kategori, emoji
4. Tap **Simpan Produk**
